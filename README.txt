# Requisitos de Librerías

Para que el código funcione correctamente, es necesario instalar las siguientes librerías:

1. **Tkinter**: Biblioteca estándar de Python para crear interfaces gráficas de usuario (GUI). Viene incluida con Python
2. **Pillow**: Biblioteca de procesamiento de imágenes de Python que permite manipular, redimensionar y modificar imágenes en diferentes formatos.

Puedes instalar estas librerías utilizando el siguiente comando:
1.  pip install tk
2.  pip install Pillow
